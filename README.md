# Pyramid Pattern Generator

A simple and interactive web-based tool that allows users to create pyramid patterns using a chosen symbol and height.

## Features
- **Custom Height:** Control how tall the pyramid will be.
- **Custom Symbol:** Use any character or symbol to build your pattern.
- **Responsive Design:** Works well on both mobile and desktop screens.

## How to Use
1. Clone this repository to your local machine.
2. Open `https://muhammadzaid-kh.github.io/Pyramid-Pattern-Generator/` in your web browser.
3. Enter the desired height (number) in the first input field.
4. Enter the symbol you want to use in the second input field.
5. Click the **Check** button to generate the pattern!
